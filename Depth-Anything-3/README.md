# Quick Start

Заходим в папку da3_streaming:
```bash
cd da3_streaming
```

Загружаем нужные веса (на cds2 веса уже скачаны, поэтому переходим к следующему шагу):
```bash
bash ./scripts/download_weights.sh
```

Собираем контейнер:
```bash
bash build.sh
```

В `da3_streaming/configs/base_config.yaml` меняем параметры `chunk_size` и `overlap`, чем больше - тем лучше.

Запускаем контейнер (в нем примонтирована вся текущая папка):
```bash
bash start.sh
```

Внутри контейнера запускаем скрипт:
```bash
python3 da3_streaming.py --image_dir <путь к папке с картинками>
```

При запуски скрипта, возможно, начнут скачиваться недостающие веса, которые затем сохранятся в кэше и при следующем запуске загружаться не будут. На некоторых серверах из-за проблем с сетью веса не скачиваются, в таком случае можно загрузить вручную папку `.cache` в папку `da3_streaming`, эту папку можно найти на cds2 по пути `/home/lazarev_aa/da3_weights/.cache`.
